Projet 2 - Cluster Kubernetes (EKS)

Déploiement d'un cluster managé AWS EKS avec CI/CD, Ingress/ALB, monitoring & observabilité.

Objectif

Construire un environnement Kubernetes de niveau production sur AWS EKS : provisioning réseau, nœuds managés, publication via Load Balancer/Ingress, et chaîne de déploiement automatisée.

Stack & Outils

  • AWS : EKS, VPC/Subnets, IAM, ALB/NLB
  • IaC : Terraform
  • Config Mgmt : Ansible
  • CI/CD : Jenkins ou GitHub Actions
  • Observabilité : Prometheus, Grafana, ELK

Fonctionnalités clés

  • Cluster EKS HA & autoscaling
  • Déploiements Rolling/Blue-Green
  • Ingress/LoadBalancer (ALB)
  • Logs & métriques centralisés
  • Gestion sécurisée des secrets
