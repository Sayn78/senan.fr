Projet 1 - Pipeline CI/CD

Déploiement d'un site HTML statique via Jenkins, SonarCloud, Docker, Ansible & Terraform sur AWS.

Objectif

Mettre en place une chaîne CI/CD de bout en bout : analyses qualité/sécurité, build d'image Docker, provisionnement de l'infrastructure, puis déploiement automatisé de l'application sur AWS.

Stack & Outils

  • CI/CD : Jenkins
  • Qualité : SonarCloud
  • IaC : Terraform
  • Config Mgmt : Ansible
  • Containers : Docker
  • Cloud : AWS (EC2, SG, IAM…)

Fonctionnalités clés

  • Build & tests automatisés
  • Analyse qualité/sécurité continue
  • Build & push image Docker
  • Provisioning infra avec Terraform
  • Déploiement via Ansible
  • Rollback simplifié
